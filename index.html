IMA SDK Plugin cho Video.js ## Giới thiệu IMA SDK Plugin cho Video.js cung cấp một tích hợp IMA SDK nhanh chóng và dễ dàng cho các cầu thủ Video.js. Khung hiện đang trong phiên bản beta, cho phép các nhà phát triển quan tâm để thử nó ra và gửi phản hồi trước khi chúng tôi hoàn tất các API và các tính năng. Để xem các plugin trong hành động, hãy kiểm tra của chúng tôi mẫu. Các tính năng ## - Dễ dàng tích hợp Google IMA SDK vào Video.js để cho phép quảng cáo trên nội dung video của bạn. ## Yêu cầu - soạn thảo văn bản yêu thích của bạn - Một trình duyệt kích hoạt JavaScript ## Bắt đầu Cách dễ nhất để bắt đầu là bằng cách sử dụng NPM. `NPM cài đặt videojs-ima` Nếu bạn không sử dụng NPM, bạn có thể tải về mã nguồn từ src / thư mục và bao gồm nó trực tiếp trong dự án của bạn. Bạn cũng sẽ cần phải tải về mã nguồn cho các videojs-contrib-quảng cáo Plugin. Trong file index.html của bạn, tạo ra một cầu thủ video.js mới và tải một (có hàng) javascript file:! `Html <html> <head> <- stylesheets phụ thuộc Load. -> <Link href = "// vjs.zencdn.net/4.5/video-js.css" rel = "stylesheet"> <link rel = "stylesheet" href = "path / to / videojs.ads.css" /> <link rel = "stylesheet" href = "path / to / videojs.ima.css" /> </ head> <body> <video id = "content_video" class = "video-js VJs-default-da" điều khiển preload = "auto" width = "YOUR_VIDEO_WIDTH" height = "YOUR_VIDEO_HEIGHT"> <source src = "PATH_TO_YOUR_CONTENT_VIDEO" type = "YOUR_CONTENT_VIDEO_TYPE" /> </ video> <! - kịch bản phụ thuộc Load -> <script src = " //vjs.zencdn.net/4.5/video.js "> </ script> <script src =" // imasdk.googleapis.com/js/sdkloader/ima3.js "> </ script> <script src =" /path/to/videojs.ads.js "> </ script> <script src =" / path / to / videojs.ima.js "> </ script> <script src =" player.js "> </ script > </ body> </ html> Trong player.js, tải các thư viện quảng cáo và thiết lập các plugin IMA: javascript var chơi = videojs ('content_video'); tùy chọn var = {id: 'content_video', adTagUrl: 'YOUR_AD_TAG'}; player.ima (tùy chọn); player.ima.requestAds (); // Trên các thiết bị di động, bạn phải gọi initializeAdDisplayContainer như là kết quả của một hành động // sử dụng (ví dụ như nút bấm). Nếu bạn không thực hiện cuộc gọi này, SDK // sẽ làm điều đó cho bạn, nhưng không phải là kết quả của một hành động của người dùng. Để biết thêm // xem các ví dụ của chúng tôi, tất cả đều được thiết lập để làm việc trên các thiết bị di động. // Player.ima.initializeAdDisplayContainer (); player.play (); Điều đó là tất cả để có nó! ## Cài đặt bổ sung Các plugin chấp nhận các thiết lập bổ sung ngoài hai cài đặt cần thiết thể hiện trong đoạn trước. Một bản tóm tắt của tất cả các thiết lập sau:javascript {id: YÊU CẦU Các id của adTagUrl chơi video của bạn: yêu cầu một URL mà trả về một adsRenderingSettings phản hồi VAST: đối tượng JSON với các quảng cáo thiết lập dựng theo quy định trong IMA SDK Documents (1). contribAdsSettings: Cài đặt bổ sung để được thông qua các plugin contrib-quảng cáo (2), được sử dụng bởi các plugin IMA này. debug: Đúng như tải các phiên bản debug của các plugin, false để tải các phiên bản không gỡ lỗi. Mặc định là false. locale: Locale cho nội địa hóa quảng cáo. Đây có thể là bất kỳ tiêu chuẩn ISO 639-1 (hai chữ) hoặc ISO 639-2 (ba chữ) mã (3). Mặc định là 'en'. nonLinearWidth: chiều rộng mong muốn của quảng cáo phi tuyến tính. Mặc định là chiều rộng cầu thủ. nonLinearHeight: chiều cao mong muốn cho quảng cáo không tuyến tính. Mặc định là 1/3 chiều cao người chơi. showCountdown: Có hay không để hiển thị các quảng cáo đồng hồ đếm ngược. Mặc định là đúng. vpaidAllowed: Dù có hay không cho phép quảng cáo VPAID. Mặc định là đúng. } `(1) [IMA SDK
